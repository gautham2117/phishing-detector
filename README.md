## рЯЪА Quick Start

**Prerequisites:** Python 3.10+, [Nmap](https://nmap.org/download.html), [Tesseract](https://github.com/UB-Mannheim/tesseract/wiki)

```powershell
# 1 вАФ Clone & enter
git clone https://github.com/your-username/mahoraga-sentinel.git
cd mahoraga-sentinel

# 2 вАФ Virtual environment
python -m venv venv
.\venv\Scripts\Activate.ps1

# 3 вАФ Install dependencies
pip install -r requirements.txt

# 4 вАФ Create database folder & .env
mkdir database
copy .env.example .env        # then edit FLASK_SECRET_KEY

# 5 вАФ Train the Random Forest model (10 seconds)
python -m backend.ml.train_url_classifier

# 6 вАФ Terminal A: start FastAPI (downloads models on first run)
python -m backend.run_fastapi

# 7 вАФ Terminal B: start Flask
python -m backend.run_flask
```

Then open **<http://127.0.0.1:5000>** вАФ select your role and enter Sentinel.
